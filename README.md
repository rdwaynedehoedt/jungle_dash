# Jungle Dash

Frontend for Jungle Dash game - React + TypeScript + Vite

## Tech Stack

- React 18 + TypeScript
- Vite
- TailwindCSS
- Zustand (state management)
- Zod (validation)
- Axios (HTTP client)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
